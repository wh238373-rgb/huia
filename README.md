# MEXC / Gate Spread Bot

Бот для моніторингу всіх спільних пар між `MEXC` та `Gate`, пошуку великих спредів і автоматичної відправки сигналів у Telegram-канал.

Що вже є:

- автозавантаження всіх спільних пар `MEXC/Gate`
- batch-скан ринку по всіх парах
- розрахунок `fair price`
- поріг сповіщення, за замовчуванням `5%`
- formatter повідомлення в стилі Telegram-поста
- Telegram notifier з `sendMessage` та `editMessageText`
- консольний dry-run режим для тесту без Telegram

## Логіка

Для кожної спільної пари базова `fair price` рахується як середнє між актуальними цінами бірж:

`fairPrice = (mexcPrice + gatePrice) / 2`

Потім бот рахує:

- `spreadPercent` між `MEXC` та `Gate`
- відхилення кожної біржі від `fair price`
- напрямок: де дешевше купувати і де дорожче продавати

Якщо `spreadPercent` більше або дорівнює порогу, бот відкриває сигнал. Далі він сам оновлює це ж повідомлення в каналі, поки спред активний. Коли спред падає нижче порогу, бот закриває сигнал редагуванням повідомлення.

## Запуск

```bash
cp .env.example .env
npm start
```

Або одним файлом:

```bash
chmod +x run-bot.sh
./run-bot.sh
```

## Налаштування

- `THRESHOLD_PERCENT` приклад: `5`
- `POLL_INTERVAL_MS` як часто оновлювати весь ринок
- `MAX_ACTIVE_SIGNALS` скільки активних повідомлень одночасно можна тримати в каналі
- `MIN_SIGNAL_UPDATE_MS` мінімальна пауза між редагуваннями одного сигналу
- `TELEGRAM_BOT_TOKEN` токен бота
- `TELEGRAM_CHAT_ID` id каналу або чату
- `TELEGRAM_THREAD_ID` опційно, якщо це forum topic
- `DRY_RUN=true` для локального тесту без відправки у Telegram
